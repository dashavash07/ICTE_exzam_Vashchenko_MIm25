# Посібник користувача: 3D-координатна система

Цей посібник допоможе вам швидко розпочати роботу з інтерактивним тренажером векторів.

## Навігація у 3D-просторі
Сцена розміщена у правій частині екрана. Для орієнтації використовуються кольорові осі:
- **Червона вісь (X)** — вісь абсцис.
- **Зелена вісь (Y)** — вісь ординат.
- **Синя вісь (Z)** — вісь аплікат.
- **Сітка** — площина XZ (підлога) для кращого просторового сприйняття.

**Управління камерою:**
- Натисніть та утримуйте ліву кнопку миші на 3D-сцені, щоб обертати камеру та роздивлятися вектори з різних кутів.

## Робота з векторами
Ліва панель призначена для введення даних та вибору операцій.

1. **Введення координат:**
   - Знайдіть блоки **Вектор a** та **Вектор b**.
   - Введіть числові значення для координат x, y та z у відповідні поля (можна використовувати десяткові дроби, наприклад, 1.5).
   
2. **Виконання операцій:**
   Натисніть одну з кнопок:
   - **Додавання (a + b):** На сцені з'являться вектори a (жовтий) та b (фіолетовий), а також результуючий вектор c (товстіший бірюзовий). Результат у числах з'явиться внизу панелі.
   - **Віднімання (a - b):** Аналогічно до додавання, будується вектор різниці.
   - **Скалярний добуток:** На сцені побудуються вектори a та b, а внизу панелі з'явиться числове значення їх скалярного добутку.

3. **Очищення:**
   При виборі нової операції або зміні координат та повторному натисканні кнопки, сцена автоматично очищується від попередніх векторів.
