# 3D-координатна система (Завдання A.1.4)

## Про проєкт
Інтерактивний VR-тренажер для вивчення просторових координат та векторів. Проєкт дозволяє користувачам будувати вектори у тривимірному просторі та виконувати базові векторні операції (додавання, віднімання, скалярний добуток).

## Функціональність
- Інтерактивна 3D-система координат (осі X, Y, Z).
- Побудова векторів за введеними координатами від початку координат (0; 0; 0).
- Візуалізація математичних операцій:
  - Додавання векторів
  - Віднімання векторів
  - Обчислення скалярного добутку

## Технології
- **HTML5 / CSS3** (Flexbox для адаптивного інтерфейсу)
- **JavaScript** (математична логіка та маніпуляції з DOM)
- **A-Frame** (WebGL фреймворк для 3D/VR сцени)
